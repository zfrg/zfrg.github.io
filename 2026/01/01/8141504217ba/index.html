TimeDisplay：全屏显示当前时间

2026年01月01日 387 字 大概 1 分钟

TimeDisplay

功能简介

这是一个基于 Python 的桌面应用程序，使用 pywebview 库构建。主要功能是在全屏窗口中实时显示当前时间，并提供一个简单的退出按钮界面。使用建议：大屏幕设备展示（如希沃白板）。

核心特点

  • 全屏时间显示：应用程序启动后以全屏模式展示，便于查看
  • 实时更新：时间每秒自动更新，确保显示准确
  • 简洁界面：包含一个专门设计的退出按钮，使用 SVG 图标增强视觉效果

使用方式

[推荐]如果你已安装了 Python：

  1. 克隆仓库（zfrg/TimeDisplay）
  1. 环境准备
    首先确保已安装必要的 Python 库：
  1. 运行应用
    运行main.pyw：
  1. 操作方法
  • 应用启动后会自动进入全屏模式
  • 屏幕中央显示当前系统时间
  • 点击左下角的方形按钮即可退出应用

如果你未安装 Python：

  1. 前往 Github Release 获取桌面端程序（Windows7+）
  2. 网页版正在开发

技术说明

  • 界面使用 HTML/CSS/JavaScript 构建，通过 pywebview 在桌面环境中渲染
  • 退出功能通过 Python 与 JavaScript 的交互实现
  • 时间更新由 JavaScript 的 setInterval 函数驱动，确保显示流畅

注意事项

  • 确保系统支持 Python 3.6 及以上版本
  • 应用依赖 pywebview 库，需要提前安装
  • 退出功能通过自定义的 Api 类实现，确保窗口正确关闭
