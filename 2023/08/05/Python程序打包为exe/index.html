Python语言简单，易于理解，有很多优点。但是，我们如果想要发布我们的程序就会有个问题，Python程序的代码往往存在.py文件或.pyw文件中，这种文件必须在安装了Python环境后才能运行。这时，我们就需要将Python程序打包为exe。

本文适合有类似困扰，且掌握Python基础知识的读者阅读

过程

准备

文本所述方法需要先安装auto-py-to-exe库。
在cmd中执行如下命令：
python.exe -m pip install auto-py-to-exe

安装过程若没有报错，基本上可以说明安装完成。

推荐将Python加入环境变量

使用

在cmd中执行auto-py-to-exe就可以了。

切记不要再auto-py-to-exe运行过程中关闭cmd终端窗口，否则都将关闭。
