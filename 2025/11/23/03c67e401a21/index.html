『Cascadia系列字体』使你编程更高效！

2025年11月23日 290 字 大概 1 分钟

介绍

这是什么？

Cascadia 是一款有趣的新代码字体，随 Windows Terminal 一同提供，现在也成为了 Visual Studio 的默认字体。——Cascadia自述文件

其实就是一款微软推出的等宽字体。

为什么选择？

主要原因有以下几点：

  1. 大厂出品，值得信赖
  2. 美观优雅，外观大气
  3. 多种格式，兼容性广

更多优势可以在Github查看（鸣谢列表中还有 Fira Code 哦）

使用

下载安装

直接访问 Github Release 发布页 下载最新版本的字体包即可。

注意事项

当然，如果你的电脑上有 Visual Studio Code、Windows Terminal 之类的软件，这些软件可能已经附带了这个字体。如果你发现你已安装了这一字体，你就可以直接使用该字体。如果你的字体版本过旧，你可以先卸载原先的旧版本，然后再前往 Github Release 发布页 以安装较新的版本。

Cascadia Mono 版本是没有连字效果的，这是该版本和 Cascadia Code 最大的区别，请根据需要选择最适合你的那个版本使用。
